
# 成绩评级程序 (fsTest) 文档

## 1. 程序概述
本程序是一个基础的 Java 控制台应用。它接收用户从键盘输入的整数成绩，根据预设的分数段区间，输出对应的成绩等级（A/B/C/D 或不及格）。

## 2. 运行环境
- **JDK 版本**：17
- **项目构建**：Maven （GroupId: `com.tyut`, ArtifactId: `dyTest`）

## 3. 逻辑规则
程序的核心判断逻辑位于 `src/main/java/fsTest.java` 的第8-20行，规则映射如下：

| 输入分数范围 | 输出等级 |
| :--- | :--- |
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| 0 - 59 | 不及格 |

## 4. 运行流程
1. **启动**: 运行 `fsTest` 类的 `main` 方法。
2. **输入**: 控制台提示“请输入成绩: ”，等待用户输入整数并回车。
3. **处理**: 程序通过 `if-else if` 语句从高到低逐级匹配分数区间。
4. **输出**: 打印匹配到的等级结果，程序结束。
